# Project Context

Project: VKACH

Description:
Fitness social network for athletes and coaches.

Stack:
- Node.js
- MySQL
- HTML
- CSS
- JavaScript

Core Features:
- Profiles
- Messages
- Training protocols
- Public training programs
- Modular profile blocks

Architecture:
Profiles are built using a block system.

Each block is independent.

Examples:
- Photo Gallery
- Services
- Reviews
- Achievements

Database:
user_blocks stores profile blocks.
Each block type has its own table.
