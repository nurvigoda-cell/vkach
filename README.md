# VKACH

VKACH is a fitness-focused social platform that combines communication, athlete profiles, and training management tools in a single ecosystem.

The goal of the project is to provide athletes, coaches, and fitness professionals with a place where they can communicate, share knowledge, publish training programs, and manage the training process without relying on multiple external services.

## Core Features

### Social Features

* User profiles
* Followers and subscriptions
* Direct messaging
* Community interaction

### Training System

* Training protocols
* Public training program sharing
* Coach-to-athlete collaboration
* Structured workout management

### Modular Profile System

Profiles are built using a block-based architecture.

Users can add, remove, reorder, and manage independent profile blocks.

Examples:

* Photo Gallery
* Services
* Reviews
* Achievements
* Education
* Before / After

New blocks can be added without modifying the profile page architecture.

### Administration

* Content moderation
* User management
* Administrative dashboard

## Project Vision

VKACH is not just a social network and not just a workout tracker.

It is a fitness ecosystem where coaches can present their expertise, publish training programs, communicate with clients, and manage the training process within a single platform.
